# 데구르르 기능 명세


## 주요 기능

- **회원 관리 기능**
    - 데구르르 회원은 이메일, 비밀번호를 통해 로그인
    - 관리자와 일반회원 존재
    - 일반회원은 정회원과 준회원 선택
    - 관리자는 회원 추가, 수정, 삭제 가능
    - 정회원, 준회원은 본인의 프로필 수정 가능
    - 프로필은 id, password, description 정보를 가짐
    - 로그인은 단순 id, password 검증으로만 진행
- **출석 관리 기능**
    - 일반 회원은 출석 여부 체크만 가능
    - 관리자는 일정 등록 가능 (default : 토요일 오전 10시)
    - 등록된 일정에 따라 모든 회원은 자신의 출석 여부 체크 가능
- **점수 관리 기능**
    - 회원은 등록된 일정을 선택하여 자신의 점수를 기록
    - 점수는 단순 스코어 입력으로 진행
    - 등록된 점수는 그래프의 형태로 표현해주고, 매 일정의 평균 점수 계산
